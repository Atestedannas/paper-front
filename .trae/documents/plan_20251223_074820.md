# 实现论文下载免费/付费功能方案

## 1. 方案概述

根据后端配置决定论文下载是免费还是付费，集成支付宝和微信支付功能，实现完整的支付流程。

## 2. 核心功能

### 2.1 下载配置管理
- 后端提供 API 获取下载配置（免费/付费）
- 前端根据配置显示不同的下载选项

### 2.2 支付流程实现
- 创建订单 API 调用
- 支付页面/弹窗设计
- 支付宝支付集成
- 微信支付集成
- 支付结果处理
- 下载权限管理

### 2.3 订单管理
- 订单创建
- 订单状态查询
- 支付回调处理
- 下载权限验证

## 3. 实现步骤

```javascript
// 下载处理函数

    // 付费下载，创建订单

      const response = await createOrder(paperId.value, 'alipay')
      orderId.value = response.data.order_id
      payModalVisible.value = true
    } catch (error) {
      ElMessage.error('创建订单失败')
    }
  } else {
    // 免费下载，直接调用下载 API
    await exportCorrectedPaper()
```

### 4.3 支付结果处理

#### 4.3.1 支付成功回调
```javascript
// 支付成功回调处理
const handlePaySuccess = async (orderId) => {
  try {
    // 查询订单状态
    const response = await queryPayResult(orderId)
    if (response.data.status === 'paid') {
      ElMessage.success('支付成功')
      // 调用下载 API
      await exportCorrectedPaper()
      // 关闭支付弹窗

    }
  } catch (error) {

  }
}
```
